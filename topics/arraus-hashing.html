Contains Duplicate

Question:

Check if an array contains any duplicates.

In One Line:

Use a set to check if any element repeats.

Simple Explanation:

We go through each number. We keep a list of ones we've already seen. If a number shows up again, we know it's a duplicate.

Mnemonics:

  • "Seen before?" → if num in seen: return True
  • "Add new" → seen.add(num)
  • "End clean" → return False

Code Implementation:

def hasDuplicate(nums):
    seen = set()
    for num in nums:
        if num in seen:
            return True
        seen.add(num)
    return False
Time Complexity: O(n) - one pass through the array
Space Complexity: O(n) - in worst case, all elements in set

Valid Anagram

Question:

Determine if two strings are anagrams (contain the same letters).

In One Line:

Use counters (hashmaps) to compare character frequencies.

Simple Explanation:

We count the letters in both words. If every letter shows up the same number of times, it's an anagram. Otherwise, it's not.

Mnemonics:

  • "Count both" → Counter(s) == Counter(t)
  • "Compare maps" → for key in counts: compare values
  • "Return result" → return True/False

Code Implementation:

def isAnagram(s, t):
    if len(s) != len(t):
        return False

    count = [0] * 26
    for i in range(len(s)):
        count[ord(s[i]) - ord('a')] += 1
        count[ord(t[i]) - ord('a')] -= 1

    for val in count:
        if val != 0:
            return False
    return True
Time Complexity: O(n) - where n is the length of the strings
Space Complexity: O(1) - fixed array size for character counts
